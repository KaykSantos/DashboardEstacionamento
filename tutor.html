Tutorial

Pequeno tutorial para utilização do sistema:

O campo de pesquisa altera as opções do campo de seleção ao lado.

Neste campo de seleção, você pode escolher o cliente para ver os dados.

Para escolher um usuário com mais velocidade, ao clicar no campo de seleção, você pode apertar o número do código do usuário (caso já o saiba), o qual já será selecionado.

Ao clicar no botão "Mostrar" os dados do usuário selecionado no campo ao lado serão exibidos.
